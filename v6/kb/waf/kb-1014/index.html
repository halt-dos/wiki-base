KB:1014

How to enable IPv6 support in listener

Problem Statement

The client requires IPv6 to be enabled on the listener level.

Solution

The above requirement can be achieved by enabling IPv6 option under listener in Haltdos WAF.

1.Go to Stack > WAF > listener name > listener settings

  1. Enable IPv6 option.

![kb-1014](/img/waf/tutorials/kbb14.png

  1. Go to Stack > Instance > Click on Instance name > Network > Ethernet. select the active interface and enable IPv6 address option and configure the settings.

  1. Click on save changes. In this way user can enable IPv6 support in their listener. User will add static/dynamic IPv6 address to listen.
