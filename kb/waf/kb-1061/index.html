How to prevent sites from JSON injection?​

Problem Statement​

Client wants to protect their sites from JSON injection attack which are protected by Haltdos solutions.

Solution​

Steps to Solve:-

The client can achieve the above requirement by configuring JSON policies.

  1. Firstly, login to WAF.

  1. Go to Apps > WAF > Listener > Profile > Profile default setting > Policy > JSON Policy > Configure the Policy > Save Changes.

  1. Now inject malicious JSON data to the respective listener.

  2. Haltdos solutions take action on the malicious JSON data as per the configuration saved.
