How to set mitigation for Path/Directory traversal attack?

Problem Statement

The user wants to set mitigation for the path traversal attack.

Solution

  1. First log into the Haltdos management portal.

  2. Go to stack > WAF > listener > profile > profile settings > signatures.

  1. Select all signature by select all button.

  1. Click on update button and select "Prod" and update the status for all the signature.

  1. Go to stack > WAF > listener > profile > profile settings >settings.

  1. Set operational settings to mitigation to mitigate the attack.

In this way user can set and the mitigation and mitigate the attack.
