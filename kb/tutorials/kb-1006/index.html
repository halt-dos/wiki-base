KB:1006

How to monitor origin servers ?

Problem Statement

The client requires to have health check for the Server Group

Solution

The client can achieve this by attaching the Monitor with Server Group

Steps to Solve

Go to Monitor page and create a monitor.

  1. Attach the Monitor to the Server Group

  1. Now go to the Incidents page to see the status of Server Group
