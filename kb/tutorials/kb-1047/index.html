How to set mitigation for OS command Injection?

Problem Statement

The user wants to set the mitigation for the OS command injection attack.

Solution

  1. First log into the Haltdos management portal.

  1. Go to stack > WAF > listener > profile > profile settings > signatures

  1. Select all signature and update status as "Prod" status.

  1. Now, go to stack > WAF > listener > profile > profile settings.

  1. Set the operational settings to mitigation.

In this way, user can set the mitigation for the OS command injection attack and mitigate the attack.
