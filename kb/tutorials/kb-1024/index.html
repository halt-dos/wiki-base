How to add a new server?

Problem Statement

The user wants to add a new server in the server group which is previously created.

Solution

  1. First log into the management console portal.

2.Go to listener > Load Balancing > Servers.

  1. Now, click on "add server" option and configure the settings.

  1. Click on save changes. In this way users can add a new server in previously created server group.
