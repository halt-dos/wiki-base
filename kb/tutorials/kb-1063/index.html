How WAF Mitigate RCE drops?

Problem Statement

The user wants to set the WAF mitigation which can mitigate RCE attack.

Solution

  1. First log into the haltdos management portal.

  1. Go to Stacks > WAF> Listener > Profile > Profile default setting > set operational mode as MITIGATION > Save Changes

Send some RCE payload on the WAF listener.

  1. Now go to WAF > Incidents. Observe the drop caused due to RCE attack
