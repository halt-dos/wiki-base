How to set rule staging from Haltdos console for managing Signatures?

Problem Statement

The client wants apply staging policy on Signature through Haltdos console.

Solution

  1. First log into the haltdos management portal.

2.Go to Stacks > WAF > Listener > Rule staging

  1. Under staging settings, user can configure staging on the basis of MINIMUM STAGING REQUESTS, MINIMUM STAGING DURATION, MAXIMUM MATCH THRESHOLD & STAGING POLICY.

  2. For monitor signatures client can follow this path Stacks > WAF > Listener > Profile > Profile default setting > Signatures
