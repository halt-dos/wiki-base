<div className='gallery'>

Protect your web applications and servers using our community edition WAF. Our community edition provides 360 degrees of website security from OWASP 10 threats, XSS, SQL and other web-based threats.  
      <br />

      In this age of internet world, web application attacks are the major targets for cyber attacks. The application serves users or requests from across the globe and hence it is difficult to monitor and protect the application from attacks. Any unwanted interuptions in the services is un-accepted and cause financial and effect the user experience. 
      <br /><br />
      <b>Web Application Firewall (WAF)</b> is essential for a web application to protect the application from such attacks.Haltdos allows every user to protect their applications from attacks and to ensure seamless user experience with 0 downtime.
